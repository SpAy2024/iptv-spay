# Mi IPTV Player

Reproductor IPTV con extractor automático de URLs M3U8.

## Características

- 📺 Canales organizados por país
- 🔍 Búsqueda de canales
- 🎨 Diseño moderno y responsive
- 🔄 Actualización automática cada 6 horas
- 🎬 Reproductor HLS integrado

## Uso

1. Visita `https://[tu-usuario].github.io/mi-iptv-player/`
2. Selecciona un canal de la lista
3. ¡Disfruta!

## Actualización manual

```bash
npm install
npm run build
